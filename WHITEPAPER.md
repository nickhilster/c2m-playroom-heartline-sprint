# Whitepaper: Heartline Sprint

## Contribution Type
Playroom kinetic-social microgame concept.

## Abstract
Heartline Sprint is a one-lane momentum game where users sculpt a runner's rhythm with tap cadence. The environment is generated from heartbeat-like waveforms, converting physical timing into architecture. Each run creates a shareable "heartline" signature.

It contributes a high-energy, short-session anchor experience to Playroom and complements ambient entries with a playful athletic mode.

## Problem Statement
Playroom requires session diversity: users should be able to switch from meditative to kinetic content without leaving the ecosystem.

## Experience Thesis
Timing is identity. Micro-rhythm can become visual storytelling.

## Core Interaction Model
- Tap or press to accelerate; release to glide.
- Perfect rhythm windows convert obstacles into ramps.
- Drift segments reward controlled under-input, not only speed.

## Session Design
- Session length: 45 to 75 seconds.
- Adaptive beat map based on first 8 user inputs.
- Finish screen summarizes rhythm consistency and style profile.

## Visual Language
- Neon cardiogram tracks become terrain.
- Runner silhouette leaves particle wake tied to tempo stability.
- Palette shifts from cool to warm as momentum increases.

## Audio Design
- Percussive click-bed synchronized to obstacle cadence.
- Bass pulse mirrors acceleration state.
- Accessibility setting replaces beat-cues with haptic-like visual flashes.

## Technical Architecture
- Deterministic procedural lane generation from seed + timing stream.
- Lightweight physics for jump arcs and drift windows.
- Input abstraction supports touch, keyboard, and gamepad.

## Accessibility
- One-button mode preserves core play loop.
- Reduced-strobe mode clamps palette flashes.
- Left-handed HUD variant available.

## Playroom Integration
- Interact style tags: run, rhythm, tap.
- Feed preview: immediate movement and readable objective.
- Share payload: heartline curve image + replay seed.

## MVP Prototype Scope
- One biome and one runner avatar.
- Single-button controls with optional advanced mode.
- End card with seed and style grade.

## Why This Matters to c2m-verse
Heartline Sprint gives Playroom a strong "one-more-run" loop while still feeling authored and artistic. It is competitive with itself, not with other people, fitting the c2m-verse ethos of expressive play.
